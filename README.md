# TrafficLens

A geospatial traffic analytics dashboard built with React, TypeScript, 
Chakra UI, Mapbox GL JS, and ApexCharts. Features an AI-powered natural 
language query bar using the Claude API.

**Live demo:** https://trafficlens-lilac.vercel.app

## Stack
- React + TypeScript (Vite)
- Chakra UI v2
- Mapbox GL JS + react-map-gl
- ApexCharts
- Claude API (Anthropic)
- Tailwind CSS v3

## Features
- Interactive Mapbox map with color-coded risk markers
- AI query bar — type natural language to filter segments
- Hourly traffic volume bar chart
- Day × hour speed heatmap
- Radial speed gauge per selected segment
- Risk rankings table
- Time filter (Today / Week / Month)
